什么是三级分销？

将推广地址发布出去，有人注册会员后会自动成为你的下线

用户 A 推荐用户B在本站充值，用户 A 获得提成

用户 B 推荐用户C在本站充值，用户 A、B 均获得提成

用户 C 推荐用户D在本站充值，用户 A、B、C 均获得提成;形成 A\B\C 三级分销人员均获得相应的提成

用户 D 推荐用户E在本站充值，这时用户A不再获得提成，只有 B\C\D 三级分销人员获得

分销如何赚钱？

• 一级分销，最高可获得直接推广会员消费金额的 50% 提成

• 二级分销，可获得下线推广佣金的 20% 提成

• 三级分销，可获得下下线推广佣金的 10% 提成

也就是，您的总收入= 一级分销的50%提成 + 二级分销的20%提成 + 三级分销的10%提成
